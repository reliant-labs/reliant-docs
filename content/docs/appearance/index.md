Personalize Reliant's look and feel to match your preferences and reduce eye strain during long coding sessions. Choose from built-in themes or create your own custom appearance.

## Theme Selection

### Choosing a Theme

1. Open **Settings** → **Appearance**
2. Browse available themes
3. Click on a theme to preview
4. Click **"Apply"** to activate

### Built-in Themes

**Light Themes**
- **Reliant Light**: Clean, minimal light theme
- **Soft Light**: Lower contrast for reduced eye strain
- **Paper**: Paper-like background with warm tones
- **High Contrast Light**: Maximum readability

**Dark Themes**
- **Reliant Dark**: Default dark theme
- **Midnight**: Deep blue-black theme
- **Monokai**: Popular developer theme
- **Dracula**: High contrast purple theme
- **Nord**: Arctic-inspired color palette

### System Theme

Follow macOS appearance:
1. Select **"System"** theme option
2. Reliant switches automatically with macOS
3. Light during day, dark at night
4. Respects macOS accent color

## Custom Themes

### Creating Themes

Build your own theme:
1. Go to **Settings** → **Appearance** → **Custom**
2. Click **"Create New Theme"**
3. Start with existing theme or blank
4. Customize colors and styles

### Theme Editor

**Color Palette**
- Background colors
- Text colors
- Border colors
- Shadow colors
- Highlight colors

**Component Styling**
- Buttons
- Input fields
- Dropdowns
- Cards
- Modals

### Importing Themes

Import community themes:
1. Download `.reliant-theme` file
2. Go to **Settings** → **Appearance**
3. Click **"Import Theme"**
4. Select the theme file
5. Preview and apply

### Exporting Themes

Share your themes:
1. Select your custom theme
2. Click **"Export"**
3. Choose export format
4. Share the file

## Quick Adjustments

### Zoom Controls

Adjust interface scale:
- **Zoom In**: `Cmd+Plus`
- **Zoom Out**: `Cmd+Minus`
- **Reset Zoom**: `Cmd+0`
- **Text Only**: Hold `Option` while zooming

### Theme Switching

Switch themes quickly:
- Use the theme selector in Settings → Appearance
- Choose from available light and dark themes
- Changes apply immediately

## Workspace Themes

### Time-Based Themes

Schedule theme changes:

## Tips

### Theme Selection

1. **Eye Comfort**: Choose based on lighting
2. **Task Type**: Different themes for different work
3. **Time of Day**: Light day, dark night
4. **Personal Preference**: What feels right

### Customization

1. **Start Simple**: Tweak existing themes
2. **Test Thoroughly**: Try in different scenarios
3. **Save Versions**: Keep theme backups
4. **Get Feedback**: Share with others

### Performance

1. **Simple Themes**: Less effects = better performance
2. **Transparency**: Can impact performance
3. **Animations**: Disable for speed
4. **High Contrast**: Often faster
